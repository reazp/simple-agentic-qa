# Simple Agentic QA

A simple agentic question-answering system using Python and AI agents.

## Files

- `agent.py` - Main agent implementation
- `simple-get-request.py` - Simple HTTP GET request utility

## Setup

1. Create a virtual environment
2. Install dependencies
3. Run the agent

## Agentic Logic Flow

Run pytest
↓
Failure?
↓
Extract assertion mismatch
↓
Call API directly
↓
Compare expected vs actual
↓
Update assertion in test file

## Usage

```bash
python agent.py
```
