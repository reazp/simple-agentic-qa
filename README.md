# Simple Agentic QA

A simple agentic question-answering system using Python and AI agents.

## Files

- `agent.py` - Main agent implementation
- `simple-get-request.py` - Simple HTTP GET request utility

## Setup

1. Create a virtual environment
2. Install dependencies
3. Run the agent

## Agentic Logic Flow

Run pytest
↓
Pytest Fails
↓
Agent Runs
↓
Parse Failure
↓
Call API To Validate
↓
Generate Fix Suggestion
↓
Create New Branch
↓
Modify Test File
↓
Commit
↓
Open Pull Request

## Usage

```bash
python agent.py
```
